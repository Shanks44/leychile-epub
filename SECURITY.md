# Security Policy

## Versiones Soportadas

| Versión | Soportada          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, por favor **no** la reportes públicamente en GitHub Issues.

En su lugar, envía un correo electrónico a **luis@aguilera.cl** con:

1. **Descripción** de la vulnerabilidad
2. **Pasos** para reproducirla
3. **Impacto potencial**
4. **Posible solución** (si la tienes)

### Qué esperar

- Recibirás confirmación de recepción en **48 horas**
- Te mantendremos informado del progreso
- Una vez corregida, serás reconocido en el CHANGELOG (a menos que prefieras permanecer anónimo)

### Alcance

Este proyecto procesa datos públicos de la Biblioteca del Congreso Nacional de Chile. Las vulnerabilidades relevantes incluyen:

- Inyección de código malicioso en ePub generados
- Exposición de datos sensibles
- Denegación de servicio
- Vulnerabilidades en dependencias

## Buenas Prácticas

Al usar este proyecto:

- Mantén tus dependencias actualizadas
- Usa entornos virtuales aislados
- Verifica los ePub generados antes de distribuirlos
- No proceses URLs de fuentes no confiables
